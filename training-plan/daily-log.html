每日登記與反思

這是飲食與訓練計畫的第二層工具頁。資料預設只存放在目前這台裝置的瀏覽器 localStorage，不會被提交到公開 GitHub repo。你可以用它快速登記，並一鍵產生要貼給 ChatGPT 的日誌格式。

安全與限制

重要：這是 GitHub Pages 靜態網站，不能提供真正的伺服器登入、真正的 Gmail 驗證碼寄送，也不能真正保護公開檔案。真正的帳號登入、密碼雜湊、Gmail OTP、資料庫加密，需要 Firebase、Supabase、Cloudflare Workers、Netlify Identity 或自己的後端。

本頁採用較安全的折衷方案：不要把每日紀錄寫進 repo；紀錄只存在你的瀏覽器本機。下方「本機密碼鎖」只防止同一台裝置被隨手打開，不等於正式資安登入。

本機密碼鎖

第一次使用時輸入帳號名稱與本機密碼，之後用同一組解鎖。密碼雜湊只存在你的瀏覽器。

狀態：尚未解鎖。

新增 / 更新每日紀錄

此功能不會寄信，只檢查本機紀錄中是否用過同一組碼。
